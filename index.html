後台登入

設定發布鑰匙（只需一次）

後台改的東西要「全世界看得到」，需要一把 GitHub 鑰匙：
1. 點 這裡開 GitHub 產生頁（要登入 GitHub）
2. Token name 隨便取（例：portfolio）
3. Repository access 選 Only select repositories → 勾 nicolas-portfolio
4. Permissions → Contents 改成 Read and write
5. 按 Generate token，把整串複製貼到下面
鑰匙只存在這台裝置的瀏覽器裡，不會出現在網站原始碼。換手機/電腦用後台時再貼一次即可。

編輯首頁文案
